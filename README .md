# ✊✋✌️ Rock Paper Scissor

A browser-based Rock Paper Scissors game built with vanilla HTML, CSS, and JavaScript. Play against the computer and track your score in a clean, interactive UI.

---

## 🎮 Demo

Open `index.html` in any modern browser — no installation or server required.

---

## 🕹️ How to Play

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Click one of the three buttons to make your move:
   - ✊ **Stone**
   - ✋ **Paper**
   - ✌️ **Scissor**
4. The computer picks its move randomly, and the result is shown instantly.
5. Scores update automatically. Hit **Play Again!** to reset.

---

## 📋 Rules

| You pick | Beats |
|----------|-------|
| ✊ Stone | ✌️ Scissor |
| ✋ Paper | ✊ Stone |
| ✌️ Scissor | ✋ Paper |

Ties result in no score change.

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/avadhootk599-crypto/rock-paper-scissor.git

# Navigate into the folder
cd rock-paper-scissor

# Open in browser
open index.html   # macOS
# or just double-click index.html on Windows/Linux
```

No dependencies. No build step. Just open and play.
